# RUNNIB

Daily functional wear for women who value comfort, safety, and peace of mind.

## Brand Story
RUNNIB은 임산부와 가임기 여성을 위한 전자파 차폐 기능성 웨어를 기획하는 브랜드입니다.  
기능만 강조한 보호용 제품이 아니라, 일상 속에서 자연스럽게 입을 수 있는 디자인과 실용성을 함께 담고자 합니다.

## Focus
- 전자파 차폐 안감 적용
- 생활형 기능성 앞치마 개발
- 여성 친화적 디자인
- 데이터 기반 신뢰 확보

## Vision
전자파 차폐 기능을 더 가깝고, 더 일상적인 방식으로 제안합니다.
